# Introduction
A web app designed for easy-disk built by Vue.js
## Project setup
```
cd app
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
